# Project Summary
The New-Year project is an interactive web application designed to celebrate the New Year with a unique digital experience. It features engaging animations, music, and user interactions, allowing users to explore festive activities and enjoy a personalized celebration.

# Project Module Description
## Functional Modules
- **ActivityPage**: Displays interactive activities related to the New Year, featuring animations and user engagement.
- **ChillZone**: A music player component that allows users to listen to curated tracks while enjoying a relaxing interface.

# Directory Tree
```
New-Year/
│
├── src/
│   ├── components/
│   │   ├── ActivityPage.tsx        # Component for interactive activities
│   │   └── ChillZone.tsx           # Music player component
│   ├── textConfig.js                # Configuration for text and labels
│   ├── music/                       # Directory for music files
│   └── musiccover/                  # Directory for music cover images
├── package.json                     # Project metadata and dependencies
└── vite.config.ts                   # Vite configuration file
```

# File Description Inventory
- **ActivityPage.tsx**: Manages user interactions and animations for holiday activities.
- **ChillZone.tsx**: Implements a music player with track selection and playback features.
- **textConfig.js**: Contains text configurations used throughout the application.
- **music/**: Holds audio files for the ChillZone component.
- **musiccover/**: Contains cover images for the music tracks.
- **package.json**: Lists project dependencies and scripts.
- **vite.config.ts**: Configures Vite for building and serving the application.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **Vite**: Build tool for front-end development.
- **TypeScript**: Superset of JavaScript for type safety.
- **CSS**: For styling and animations.

# Usage
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sisirtmg07/New-Year.git
   cd New-Year
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Build the project**:
   ```bash
   npm run build
   ```

4. **Run the development server**:
   ```bash
   npm run dev &
   ```

5. **Access the application**: Open your browser to view the application.
